# 📊 All For One 자산관리

한국 주식 포트폴리오를 한 파일로 관리하는 **단일 HTML 자산관리 도구**입니다.  
설치 불필요 · 서버 불필요 · 브라우저 자동저장

---

## ✨ 주요 기능

| 기능 | 설명 |
|------|------|
| **현재 포트폴리오** | 계좌/그룹별 보유 종목 관리, 평가손익·수익률 자동 계산 |
| **실시간 종가 조회** | 국내주식·ETF·미국주식 현재가 자동 갱신 |
| **자산 추이 분석** | 날짜별 자산 변동 그래프, TWR / 단순 수익률 선택 |
| **매매일지 자동 계산** | CSV 업로드 → 일별 실현손익·수수료·세금 자동 계산 |
| **다계좌 관리** | ISA · 연금 · 퇴직연금 · 일반계좌 분리 관리 |
| **미국주식 이중 손익** | 원화 손익 + 달러 손익 동시 표시, 계좌별 매도 우대율 설정 |
| **백업 / 복원** | 전체 데이터를 CSV로 내보내거나 불러오기 |
| **브라우저 자동저장** | localStorage 기반, 새로고침해도 데이터 유지 |

## 🚀 시작하기

### 1. 파일 실행
`거의다되는_자산관리_v2.html` 파일을 **크롬(Chrome) 브라우저**로 열면 바로 사용 가능합니다.

> 권장 브라우저: Chrome ✅ / Edge ✅ / Safari·Firefox ⚠️ 부분 지원 / IE ❌ 미지원

### 2. 실시간 종가 조회 설정 (선택)
기본 포트폴리오 입력은 인터넷 없이도 가능합니다.  
**실시간 종가 조회 및 매매일지 자동 계산**을 사용하려면 아래 두 가지가 필요합니다.

#### ① CORS 확장 프로그램 설치
브라우저 보안 정책(CORS)으로 인해 외부 시세 API를 직접 호출할 수 없습니다.  
크롬 웹스토어에서 **「Allow CORS: Access-Control-Allow-Origin」** 확장을 설치 후 ON으로 활성화합니다.

> ⚠️ 보안 주의: 자산관리 사용 후에는 반드시 확장 프로그램을 **OFF**로 되돌리세요.  
> 특히 인터넷 뱅킹·결제 시에는 꼭 끄고 사용하세요.

#### ② 공공데이터포털 API 키 발급 (무료)
1. [data.go.kr](https://www.data.go.kr) 접속 후 회원가입
2. 아래 두 가지 API 활용신청
   - `금융위원회_주식시세정보` — 국내 주식 종가 조회용
   - `금융위원회_증권상품시세정보` — ETF 종가 조회용
3. 마이페이지 → 오픈API → 개발계정에서 **일반 인증키(Encoding)** 복사
4. 앱 실행 후 **「자산 추이 분석」 탭 → API 키 입력 후 저장**

> 🕐 발급 직후 1~2시간 후 사용 가능합니다.  
> 🔑 API 키는 절대 외부에 공개하지 마세요.

## 📁 CSV 입력 형식

### 매매일지 CSV
매매 내역을 CSV 파일로 업로드하면 실현손익·수수료·세금을 자동 계산합니다.

| 열 | 내용 | 예시 |
|----|------|------|
| 날짜 | YYYY-MM-DD | `2025-01-15` |
| 종목코드 | 6자리 코드 | `005930` |
| 종목명 | 종목 이름 | `삼성전자` |
| 구분 | 매수 / 매도 | `매수` |
| 수량 | 거래 주수 | `100` |
| 단가 | 1주당 체결가 | `65000` |
| 수수료 | 거래 수수료 (0이면 자동계산) | `24` |

- 수수료가 `0`이면 `0.0036396%`로 자동 계산됩니다.
- 개별주식 증권거래세: 2026년~ 0.20% / 2025년 이전 0.15% (ISA·퇴직연금 면제)

### 원금내역 CSV
| 열 | 내용 | 예시 |
|----|------|------|
| 날짜 | YYYY-MM-DD | `2024-01-01` |
| 금액 | 입금(+) / 출금(-) | `1000000` |
| 메모 | 비고 (선택) | `초기 입금` |

### 배당금/이자 CSV
| 열 | 내용 | 예시 |
|----|------|------|
| 날짜 | YYYY-MM-DD | `2025-03-20` |
| 금액 | 세후 수령액 | `15000` |
| 메모 | 비고 (선택) | `KODEX S&P500 분배금` |

## 🌐 데이터 출처

| 종류 | 출처 |
|------|------|
| 국내 주식 종가 | 네이버 금융 API → 공공데이터포털 KRX API (fallback) |
| 국내 ETF 종가 | 공공데이터포털 `금융위원회_증권상품시세정보` |
| 미국 주식 종가 | Yahoo Finance → 네이버 해외주식 API (fallback) |
| USD/KRW 환율 | 네이버 금융 환율 API |

> 앱은 **KRX 정규장 종가** 기준입니다. NXT(넥스트레이드) 종가 및 시간외 단일가는 지원하지 않습니다.

## 🔒 개인정보 & 보안

- 모든 데이터는 **사용자 브라우저 내 localStorage에만 저장**되며 외부 서버로 전송되지 않습니다.
- API 키는 브라우저 로컬에만 저장됩니다. GitHub 등 공개 장소에 업로드하지 마세요.
- 이 저장소에는 API 키, 개인 투자 데이터가 포함되어 있지 않습니다.
